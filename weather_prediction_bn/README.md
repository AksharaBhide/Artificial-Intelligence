# Bayesian Network for Weather Prediction

A Python implementation of a Bayesian Network for probabilistic weather forecasting using Directed Acyclic Graphs (DAG) and Bayesian inference.

## Features

- **Bayesian Network Structure**: DAG representing relationships between weather factors
- **Probabilistic Inference**: Calculate weather probabilities based on observed conditions
- **Multiple Visualizations**: Various charts and graphs to understand predictions
- **Interactive Mode**: Make predictions based on your inputs
- **Synthetic Data Generation**: Generate realistic weather data for testing
